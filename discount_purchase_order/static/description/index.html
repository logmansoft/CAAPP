Discount On Purchase Order

Discount on Purchase Order module facilitates the user to apply discount on purchase order both orderline wise and globally on the whole order. Discount can be successfully applied on tax included as well as tax excluded amount. This module also allows the user to manage Purchase order and invoice report with the discount applied.

Features:

  • Manage The Discount Type : Fixed/Percent.
  • Apply A Disocunt On The Orderline Of Fixed/Percent Type.
  • Apply A Global Discount On A Purchase Order Of Fixed/Percent Type.
  • Apply The Global Discount On Order's Total Tax Included/Excluded Amount.
  • Manage The Purchase Order Report With The Applied Discount.
  • Manage The Invoice Report With The Applied Discount.
  • Manage Separate Account for Global Discount.

Admin View

Configuration

Enable The Global Discount Setting To Apply Global Discount On The Purchase Order
Choose The Option For Applying Global Discount On Order's Total Tax Included/Excluded Amount
Select the Discount Account to use in journal entry for global discount

Apply Discount On Purchase Order

Select The Discount Type : Fixed/Percent.
Enter The Discount Amount.
On The Basis Of Discount Type, Final Discount Amount Will Be Computed.
Total Discount, Discount On Orderlines and Global Discount Will Appear Before Order Total.

Invoice

Discount Type Is Managed In The Order Invoice.
Total Discount Is Also Managed In Order Invoice.

Journal Entry

Journal Entry for global discount.

Purchase Order Report

Purchase Order Reports Are Managed With Discount Type.
Purchase Order Reports Are Managed With Total Discount Amount.
